## How to use
You should install `nodejs` and `npm` libs first, then install `fand` globaly:

```terminal
npm install fand -g
```

And just it. now you can start searching:

```terminal
fand سعید
# سد
# سی
# عدس
# عید
# سعی
# عدسی
# سعید
# سعدی
# دیس
# سید
```

## Write output to file?

```terminal
fand سلام -o ./result.txt
# /home/nainemom/Projects/fand/result.txt created!
```
